# Starter kit for blog with elm-pages

## Demo

[https://elm-pages-blog-template.netlify.app/](https://elm-pages-blog-template.netlify.app/)

## Goals

- easy to start with
- easy to style and change
- prefer elm packages over JS
- nice default layout that is 
  - accessible
  - responsive
